## About
This landing page is inspired by the **THE SKIN MAKER2 game** (Horror Series, 2025).  
It demonstrates modern **Flexbox & Grid layouts**, **responsive design**, **dark/light modes**, and **interactive UI elements** like hamburger menus, buttons, and hover effects.

---

## Features
- Fully responsive (desktop & mobile)
- Light and dark mode toggle
- Hamburger menu with animated sidebar
- Modern gradient accents, glassmorphism cards
- Smooth animations and transitions
- SEO-friendly HTML structure
- Organized CSS using Flexbox and Grid
- Modular JavaScript for menu & mode toggles
